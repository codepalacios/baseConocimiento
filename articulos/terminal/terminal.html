Terminal y línea de comandos

La terminal

Es una interfaz gráfica que simula una línea de comandos. Cuando hablamos de una línea de comandos nos referimos a una shell. En informática, el shell o intérprete de órdenes o intérpetre de comandos es el programa informático que provee una interfaz de usuario para acceder a los servicios del sistema operativo.
