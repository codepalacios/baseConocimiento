Terminal y línea de comandos

La terminal

Es una interfaz gráfica que simula una línea de comandos. Cuando hablamos de una línea de comandos nos referimos a una shell (intérprete de órdenes o comandos) que es el programa que permite al usuario acceder a los servicios del sistema operativo (toma comandos y los pasa al sistema operativo para hacer algo).

Un comando es: un programa que se puede ejecutar desde la terminal. Este puede recibir parámetros y opciones.

¿Por qué aprender a usar la terminal?

  • Flexibilidad
  • Velocidad
  • No siempre se cuenta con una interfaz gráfica

Tipos de Shells:

  • Bourne Shell
  • Bash Shell
  • Z Shell
  • C Shell
  • Korn Shell
  • Fish Shell
  • PowerShell
